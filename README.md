# Тест - Предсказание

Тест предлагает вам получить предсказание. Выбор ответа из 2 и 4 вариантов, ввод своей даты рождения и валидация этих полей, экран загрузки с крутящимся спинером, проигрывание музыки с анимацией прогресса воспроизведения, вывод fetch-запроса по нажатию кнопки "Позвонить и прослушать". Для проверки данных в полях ввода используются HTML5-атрибуты и JS-свойство ValidityState. 

Технологии:
* JS 
* css анимация 
* fetch
* flexbox

Методы:
* ООП 
* БЭМ
* Валидация
* Адаптив

Планируется разбить код на классы и сделать его элементы более независимыми.
